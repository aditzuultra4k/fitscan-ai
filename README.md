# FitScan AI

FitScan AI is a premium fitness and nutrition mobile app built with React Native, Expo, TypeScript, and Expo Router.

## MVP Features

- Login and signup screens
- Dashboard with calories, macros, shortcuts, and daily meal summary
- Calorie tracker starting from zero
- Manual food entry by Breakfast, Lunch, Dinner, and Snacks
- AI food scanner placeholder with editable macro estimates
- Barcode scanner placeholder
- Meal history
- Profile and settings
- Privacy Policy screen
- Firebase Auth, Firestore, and Storage integration scaffold

## Development

```bash
npm install
npm run typecheck
npm run web
```

## App Store Notes

Before submitting to the App Store, add a production icon, splash screen, hosted Privacy Policy URL, Terms of Use, Firebase environment variables, and a clear disclaimer that AI nutrition estimates are informational and not medical advice.
